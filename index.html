How to play:

Time Attack: Type as many words as possible before the time runs out! You score 1 point for each correct word typed.

Countdown: Start with a limited amount of time. Each correct word you type adds time to the counter and 1 point to your score. Type fast to stop the counter from reaching 0! Challenge yourself with 3 Difficulties!
